# nightbasketball
"Ночной баскетбол". Простая игра для первого урока Scratch

Атмосферная игра с элементами хоррора, с привидением.
Фея защищает детей от привидения. Фея движется вертикально, управляется мышкой. Синхронно с ней вертикально движется девушка с битой на противоположной стороне сцены. Девушка с битой уменьшает размер привидения. Чтобы выиграть, надо дать уменьшенному привидению вылететь за сцену, пролетев над или под группой детей.
